Links

What is a URL?

  • Uniform Resource Locator
  • a unique address for a document
  • the document to which it is linked may be: another HTML document, an image...
  • example - protocol://servername/path/filename - https://www.thomasmore.be/images/campus.jpg

Definitions

An absolute URL
The entire address from the protocol (https) to the domain name (www.thomasmore.be) and includes the location within your website in your folder system - https://www.thomasmore.be/images/campus.jpg
A relative URL
Does not use the full web address and only contains the location following the domain - images/campus.jpg The link you add has to be on the same site and has to be part of the same root domain

Example

<a href="https://www.google.com">Link to Google</a>
